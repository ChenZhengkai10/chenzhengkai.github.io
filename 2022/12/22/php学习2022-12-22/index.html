三元运算

三元运算符的格式：(expr1) ? (expr2) : (expr3)
解释：如果条件”expr1”成立，则执行语句”expr2”,否则执行”expr3”

例：

同时，也可以用if来写一段作用相同的代码

C语言中，条件的符号用的也是 ? :
他的用法是 A?B:C
意思是：若A为非0则值为B，否则为C
php和C中这个符号的判断顺序还是一样的

if_else

if else的用法和语法大致都和C一样

if：

if（条件）
{
条件成立时要执行的代码;
}

if else

if (条件)
{
条件成立时执行的代码;
}
else
{
条件不成立时执行的代码;
}

if ifelse else

if (条件)
{
if条件成立时执行的代码;
}
elseif (条件)
{
elseif条件成立时执行的代码;
}
else
{
条件不成立时执行的代码;
